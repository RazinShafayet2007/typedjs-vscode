# TypedJS for VS Code

Language support and validation for **TypedJS**.

## Features

This extension provides rich language support for the TypedJS programming language.

- **Syntax Highlighting**: Colorful and readable code for `.js` files when using the TypedJS language mode.
- **Validation**: Integrated linting to catch errors early.
- **Snippets**: Productivity-boosting code snippets.

## Requirements

- VS Code 1.80.0 or newer.

## Extension Settings

This extension contributes the following settings:

* `typedjs.enable`: Enable/disable this extension.
* `typedjs.trace.server`: Traces the communication between VS Code and the language server.
